# 🍱 MealBell
### Campus Food Pickup Notification iOS App

MealBell은 캠퍼스 내 학식 주문 과정에서 발생하는 대기 불편을 줄이기 위해 개발된 iOS 애플리케이션입니다.

사용자는 메뉴를 확인하고 주문할 수 있으며, 음식 준비 완료 시 알림을 받아 효율적으로 음식을 픽업할 수 있습니다.  
또한 예상 픽업 시간을 제공하여 사용자가 대기 시간을 보다 효율적으로 관리할 수 있도록 도와줍니다.

---

# 📱 Project Overview

- **Project Name** : MealBell
- **Platform** : iOS
- **Development Environment** : Xcode
- **Language** : Swift
- **Framework** : SwiftUI

MealBell은 캠퍼스 학식 주문 과정에서 발생하는 불편함을 해결하기 위한 모바일 애플리케이션입니다.  
주문 상태 확인, 픽업 시간 안내, 준비 완료 알림 기능을 통해 사용자 경험을 개선하는 것을 목표로 합니다.

---

# ✨ Features

## 📋 Menu Browsing
- 캠퍼스 식당 메뉴 목록 확인
- 메뉴 이름, 가격, 설명 제공

## 🛒 Food Ordering
- 메뉴 선택 및 주문
- 수량 선택 기능

## 📊 Order Status Tracking
- 주문 접수
- 조리 중
- 준비 완료
- 픽업 완료

## ⏱ Pickup Time Estimation
- 예상 대기 시간 표시
- 예상 픽업 시간 안내

## 🔔 Ready Notification
- 음식 준비 완료 시 알림 제공
- 사용자가 앱을 계속 확인하지 않아도 픽업 가능

## 📜 Order History
- 이전 주문 내역 확인
- 주문 시간 및 메뉴 정보 제공

---

# 🏗 App Architecture

MealBell은 iOS 애플리케이션 중심의 구조로 설계되었습니다.

iOS App
 ├─ Menu List
 ├─ Order Management
 ├─ Pickup Time
 ├─ Notification
 └─ Order History

본 프로젝트는 iOS 앱 기능 구현에 초점을 맞추기 위해 로컬 데이터와 더미 데이터를 기반으로 프로토타입 형태로 개발되었습니다.
